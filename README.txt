Tower Defense Unity - Protótipo por Tales Mariano

Desenvolvido utilizando Unity 5.3.2f1


Para rodar o jogo, inicie a cena CenaPrincipal na pasta Assets.

Para se ganhar o jogo, deve se juntar 200 moedas e comprar a vitória no botão "Ganhar o Jogo".

Se perde o jogo caso 10 minions alcancem o outro lado do mapa.


Controles:

Se utiliza o mouse para controlar o jogo

Para movimentar a câmera, deve-se clicar nas setas posicionadas na parte inferior da tela.

Para posicionar uma torre, deve-se clicar num botão de torre ao lado direito da tela, e depois clicar no mapa na posição que deseja posiciona-la.

Torres não podem ser posicionadas de maneira que bloqueiem o caminho dos minions a chegada.

Botões não podem ser clicados caso não se tenho moedas suficientes.

Passar o mouse por cima de uma torre mostra o seu alcance.

Para lançar uma wave de inimigos, deve-se clicar no botão inferior esquerdo escrito "Próxima Wave".

As setas no topo das estruturas mostram a direção por onde os minions saem e entram.

Ao Fim do jogo, tanto vitória como derrota, basta apertar qualquer botão para reiniciar o jogo.
